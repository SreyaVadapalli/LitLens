## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React |
| Backend | FastAPI |
| LLM | Claude API (Anthropic) |
| Embeddings | SPECTER (allenai/specter) |
| Vector DB | ChromaDB |
| PDF Parsing | PyMuPDF |
| Literature Search | NCBI PubMed API (Biopython) |
| Protocol | MCP (Model Context Protocol) |
